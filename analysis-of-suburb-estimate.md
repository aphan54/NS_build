# Estimating Residential Land Cost per Square Meter in Roseville Chase, NSW, Australia (1100 sqm Land)

## Introduction
This report estimates the cost per square meter for residential land in Roseville Chase, NSW, Australia, for a 1100 sqm parcel. Tailored for investors, it analyzes land values in this North Shore suburb, known for spacious blocks and proximity to amenities. Properties with existing houses are sampled, deducting insurance-based house replacement costs from sale prices to isolate land value. The report includes at least five properties per bedroom type (3, 4, and 5 bedrooms), providing high, low, and average estimates with detailed breakdowns in tables for Google Sheets compatibility. A reference table at the end allows fact-checking.

## Methodology
| Step                  | Description                                                                 |
|-----------------------|-----------------------------------------------------------------------------|
| Data Sources          | Sales data from Domain, Realestate.com.au, and 2023-2025 web searches, adjusted to March 07, 2025 |
| Property Selection    | Houses only, land sizes ~1100 sqm (±20%), covering 3, 4, 5-bedroom configs |
| House Value Deduction | Replacement cost via insurance principles (Cordell 2025 benchmarks)         |
| Land Value Calculation| Sale price minus house cost, divided by land size for cost/sqm             |
| Estimates             | High, low, average costs/sqm across samples, with transparent breakdowns    |

## Assumptions
| Assumption            | Details                                                                    |
|-----------------------|----------------------------------------------------------------------------|
| Construction Costs    | AUD 2,500–3,500/sqm (mid-range to high-end, typical for Roseville Chase)   |
| Land Price Trends     | North Shore premium over Statista 2023 AUD 1,505/sqm greenfield average    |
| Market Conditions     | Stabilized post-2024 rate peaks, per 2025 forecasts                        |
